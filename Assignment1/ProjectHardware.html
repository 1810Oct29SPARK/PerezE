Components needed for machine vision

  • Servos
  • Material for body (acrylic)
  • Computer with Matlab
  • Camera
  • Microcontroller (PSoC 5)
  • Wires
  • Electromagnet
  • 12V (if needed)


Setup

  1. Build robotic arm
  2. Connect camera to computer
  3. Connect PSoC 5 to computer
  4. Connect wires to servos
  5. Run code
  6. Once computer takes first picture, place contianer and screw
  7. Continue with program
  8. Profit
