Why do dropdown menus always have to be JavsScript powered? Some CSS will make this simple solution. The only downside is that the height on hover has to be adjusted manually (or by php): the amount of languages multiplied by 40 minus 1.

If you have a lot of languages, you may consider using overflow: scroll;
Unfortunately, windows has very ugly scrollbars, so you may stiill want to use some JavaScript (jQuery plugin) to have nice and small scrollbars (like the panels of Codepen).
